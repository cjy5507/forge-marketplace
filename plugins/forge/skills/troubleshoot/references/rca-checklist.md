# RCA Checklist

Use this reference during `forge:troubleshoot`.

## Must-have outputs

- Reliable reproduction or explicit intermittent note
- At least two competing hypotheses
- Evidence for and against each hypothesis
- Confirmed root cause
- Minimal fix
- Impact and regression assessment

## Do not do

- Fix before root cause confirmation
- Keep only one hypothesis
- Propose a broad refactor when a bounded fix exists
- Skip regression planning
