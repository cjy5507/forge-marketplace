# Design Deliverables

Use this reference for the complete design output checklist.

## CTO output

- `.forge/design/architecture.md`
- `.forge/code-rules.md`
- `.forge/contracts/*.ts`
- `.forge/evidence/*`

## Designer output

- `.forge/design/components.md`
- `.forge/design/tokens.json`
- Wireframes or structured screen descriptions

## Cross-review checks

- Feasibility of interactions and animations
- Contract completeness for UI boundaries
- Responsive coverage
- Accessibility requirements
