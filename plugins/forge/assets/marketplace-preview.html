Control plane

  • Stop guard blocks premature exit while work is still active
  • Session start and end persist Forge runtime state
  • Subagent lifecycle is written into `.forge/runtime.json`

Quality gates

  • Code writes are denied if spec, design, rules, or contracts are missing
  • Post-write reminders reinforce contracts and code-rule checks
  • A/B evaluation templates measure harness value vs baseline

Roles

  • CEO, PM, CTO, Lead Dev, QA, Security, Troubleshooter
  • Progressive disclosure via references keeps prompts lean
