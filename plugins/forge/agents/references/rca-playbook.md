# RCA Playbook

Use this reference when a Forge issue needs structured diagnosis.

## Diagnosis phases

1. Reproduce
2. Generate competing hypotheses
3. Gather evidence
4. Verify or refute each hypothesis
5. Confirm root cause
6. Propose the minimal fix
7. Assess impact and regression risk

## RCA report structure

- Issue
- Symptoms
- Reproduction
- Hypotheses considered
- Root cause
- Causal chain
- Proposed fix
- Impact analysis
- Prevention

The confirmed root cause must explain all observed symptoms. If it does not, keep investigating.
